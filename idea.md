Проект: "Веб - сайт для учёта посещаемости студентов с использованием QR-кодов". 
 Подробный функционал платформы:

---

## **1. МОДУЛЬ АУТЕНТИФИКАЦИИ И АВТОРИЗАЦИИ**

### Регистрация и вход:
- Вход через логин/пароль
- Двухфакторная аутентификация (опционально)
- Восстановление пароля через email

### Роли пользователей:
- **Администратор** - полный доступ к системе
- **Преподаватель** - управление своими занятиями
- **Студент** - отметка посещаемости и просмотр статистики
- **Декан/Завкафедрой** - доступ к отчётам по своему подразделению

---

## **2. ЛИЧНЫЙ КАБИНЕТ ПРЕПОДАВАТЕЛЯ**

### Управление занятиями:
- **Создание занятия:**
  - Выбор предмета из списка
  - Указание типа занятия (лекция, практика, лабораторная)
  - Выбор группы/групп студентов
  - Дата и время начала/окончания
  - Номер аудитории
  - Примечания (тема занятия)

- **Редактирование занятий:**
  - Изменение времени, аудитории
  - Отмена занятия с уведомлением студентов
  - Перенос занятия

- **Шаблоны расписания:**
  - Создание повторяющихся занятий (каждую неделю)
  - Автоматическое создание занятий на семестр

### Генерация и управление QR-кодами:
- **Автоматическая генерация:**
  - Уникальный QR-код для каждого занятия
  - Код содержит зашифрованную информацию (ID занятия, timestamp, hash)

- **Настройки активности:**
  - Время начала действия (за 10 минут до начала пары)
  - Время окончания действия (15-20 минут после начала)
  - Возможность продлить время действия
  - Ручная деактивация кода

- **Отображение QR-кода:**
  - Полноэкранный режим для проектора
  - Печать QR-кода
  - Отправка кода в чат группы
  - Обновление кода в реальном времени (защита от фото)

### Журнал посещаемости:
- **Список студентов:**
  - Полный список группы с фото
  - Отметка присутствующих (зелёная галочка)
  - Отметка отсутствующих (красный крестик)
  - Отметка опоздавших (жёлтый знак)
  - Время сканирования QR-кода каждым студентом

- **Ручная корректировка:**
  - Добавление отметки вручную с комментарием (причина)
  - Удаление ошибочной отметки
  - Отметка по уважительной причине (с прикреплением справки)

- **Фильтры и поиск:**
  - Поиск по ФИО студента
  - Фильтр по статусу (присутствовал/отсутствовал)
  - Фильтр по дате, предмету, группе

### Статистика и отчёты:
- **По занятию:**
  - Процент явки на конкретное занятие
  - Список опоздавших
  - Сравнение с предыдущими занятиями

- **По студенту:**
  - Общий процент посещаемости по предмету
  - Количество пропусков/опозданий
  - График динамики посещаемости
  - История всех посещений

- **По группе:**
  - Рейтинг студентов по посещаемости
  - Средний процент посещаемости группы
  - Студенты группы риска (< 50% посещений)

- **Экспорт данных:**
  - Excel-таблица с посещаемостью
  - PDF-отчёт для деканата
  - CSV для дальнейшей обработки
  - Настройка периода экспорта (неделя, месяц, семестр)

### Уведомления:
- **Напоминания:**
  - О предстоящем занятии (за 1 час)
  - О необходимости создать QR-код
  
- **Оповещения:**
  - Когда студент отсканировал код
  - О низкой явке на занятие (< 60%)

## **3. ЛИЧНЫЙ КАБИНЕТ СТУДЕНТА**

### Главная страница:
- **Актуальные занятия:**
  - Список занятий на сегодня
  - Статус (ожидается, идёт сейчас, завершено)
  - Кнопка "Отметиться" для активных занятий

- **Быстрая статистика:**
  - Общий процент посещаемости
  - Количество пропусков за неделю/месяц
  - Предметы с низкой посещаемостью (предупреждение)

### Сканирование QR-кода:
- **Способы сканирования:**
  - Встроенный сканер в веб-интерфейсе (доступ к камере)
  - Через мобильное приложение
  - Ручной ввод кода (при проблемах с камерой)

- **Процесс отметки:**
  - Наведение камеры на QR-код
  - Автоматическое распознавание
  - Подтверждение отметки (вибрация, звук, визуальное)
  - Отображение информации о занятии

- **Валидация:**
  - Проверка действительности QR-кода
  - Проверка времени (код активен?)
  - Проверка местоположения (геолокация, опционально)
  - Защита от повторного сканирования

- **Обработка ошибок:**
  - "QR-код недействителен"
  - "Время сканирования истекло"
  - "Вы уже отметились на этом занятии"
  - "Вы не записаны на это занятие"

### Расписание:
- **Календарь занятий:**
  - Просмотр по дням/неделям/месяцам
  - Цветовая индикация (посещено/пропущено/предстоит)
  - Информация о занятии при клике

- **Фильтры:**
  - По предмету
  - По преподавателю
  - По типу занятия

### История посещений:
- **Журнал:**
  - Список всех посещённых занятий
  - Дата, время, предмет, преподаватель
  - Статус (присутствовал, опоздал, пропустил)
  - Время сканирования QR-кода

- **Поиск:**
  - По дате
  - По предмету
  - По статусу

### Статистика:
- **Общая:**
  - Процент посещаемости за семестр
  - Количество посещённых/пропущенных занятий
  - Среднее количество пропусков в неделю

- **По предметам:**
  - Таблица со всеми предметами
  - Процент посещаемости по каждому
  - Количество пропусков/опозданий
  - График динамики

- **Графики и визуализация:**
  - Круговая диаграмма посещаемости
  - Линейный график по неделям
  - Сравнение с средним по группе

### Уведомления:
- **Push-уведомления (в приложении):**
  - Напоминание о предстоящем занятии (30 мин до начала)
  - "Пара началась, отметьтесь!"
  - "QR-код скоро перестанет действовать"

- **Email-уведомления:**
  - Еженедельный отчёт о посещаемости
  - Предупреждение о критически низкой посещаемости
  - Уведомление о пропуске важного занятия

### Профиль:
- Личные данные (ФИО, группа, курс)
- Фотография
- Контактная информация
- Настройки уведомлений

## **4. ПАНЕЛЬ АДМИНИСТРАТОРА**

### Управление пользователями:
- **Преподаватели:**
  - Добавление нового преподавателя
  - Редактирование данных
  - Назначение предметов
  - Деактивация аккаунта
  - Импорт списка из Excel/CSV

- **Студенты:**
  - Массовое добавление (импорт из файла)
  - Распределение по группам
  - Редактирование профилей
  - Перевод в другую группу
  - Отчисление (архивирование данных)

- **Группы:**
  - Создание учебных групп
  - Назначение старосты
  - Привязка к курсу и факультету
  - Объединение/разделение групп

### Управление учебным процессом:
- **Предметы:**
  - Создание предметов (название, код, кафедра)
  - Назначение преподавателей на предметы
  - Привязка предметов к группам
  - Количество часов (лекции/практика)

- **Расписание:**
  - Импорт расписания из внешних систем
  - Массовое создание занятий на семестр
  - Редактирование расписания
  - Управление аудиторным фондом

- **Семестры:**
  - Создание учебного года и семестров
  - Установка дат начала/окончания
  - Календарь праздников и выходных
  - Сессии и пересдачи

### Отчёты и аналитика:
- **Общая статистика:**
  - Средняя посещаемость по университету
  - Топ и худшие группы по посещаемости
  - Статистика по факультетам/кафедрам
  - Динамика посещаемости по месяцам

- **Детальные отчёты:**
  - По преподавателю (явка на его занятиях)
  - По предмету
  - По группе
  - По студенту
  - Сравнительный анализ

- **Экспорт:**
  - Отчёты для деканата
  - Статистика для ректората
  - Данные для стипендиальной комиссии

### Настройки системы:
- **Параметры посещаемости:**
  - Время действия QR-кода (мин)
  - Порог для опоздания (мин)
  - Минимальный процент для допуска к экзамену

- **Безопасность:**
  - Включение геолокации
  - Радиус разрешённой зоны (метры)
  - Частота обновления QR-кода
  - Ограничение на количество попыток сканирования

- **Уведомления:**
  - Настройка шаблонов писем
  - Расписание отправки отчётов
  - Push-уведомления

- **Интеграции:**
  - API для внешних систем
  - Подключение к системе электронных журналов
  - Интеграция с LMS университета

### Логи и мониторинг:
- История действий пользователей
- Журнал изменений в базе данных
- Статистика использования системы
- Технические ошибки и инциденты

## **5. ДОПОЛНИТЕЛЬНЫЕ МОДУЛИ**

### Защита от мошенничества:
- **Геолокация:**
  - Проверка местоположения при сканировании
  - Допустимый радиус от аудитории (50-100м)
  - Предупреждение при сканировании вне кампуса

- **Динамические QR-коды:**
  - Автоматическое обновление кода каждые 30 сек
  - Защита от фотографирования и пересылки

- **IP-адреса:**
  - Логирование IP при сканировании
  - Определение подозрительной активности (много сканирований с одного IP)

- **Лимиты:**
  - Одна отметка = один студент
  - Блокировка повторного сканирования
  - Капча при подозрительной активности

### Интеграции:

- **С мессенджерами:**
  - Telegram-бот для уведомлений
  - Отправка QR в группы в WhatsApp

### Система справок и документов:
- **Для студентов:**
  - Загрузка справок об уважительной причине
  - Прикрепление к конкретному пропуску
  - История загруженных документов

- **Для преподавателей:**
  - Просмотр и проверка справок
  - Подтверждение/отклонение
  - Комментарии к документам

### Аналитика и BI:
- **Дашборды:**
  - Интерактивные графики
  - Фильтры по разным параметрам
  - Экспорт в различные форматы

- **Прогнозирование:**
  - Предсказание будущей посещаемости
  - Выявление студентов группы риска
  - Рекомендации по улучшению явки

### Система мотивации:
- **Рейтинги:**
  - Топ студентов по посещаемости
  - Топ групп
  - Достижения и бейджи

- **Поощрения:**
  - Бонусы к стипендии за 100% посещаемость
  - Возможность получить автомат
  - Грамоты и сертификаты
